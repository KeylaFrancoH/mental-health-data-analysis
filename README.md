## Dataset

This project uses the **Mental Health Dataset** available on Kaggle, which contains demographic, behavioral, and mental health-related variables for exploratory data analysis and visualization.

Dataset source:
https://www.kaggle.com/code/mahmoudweso/mental-health-dataset/input?select=Mental+Health+Dataset.csv

The dataset was used to explore patterns related to mental health indicators, demographic characteristics, and potential factors associated with mental well-being through statistical analysis and data visualization techniques.

# Mental Health Data Analysis

Data Science project focused on the analysis and visualization of mental health-related data obtained from Kaggle to identify trends, patterns, and insights through exploratory data analysis (EDA).

## Overview

This project explores mental health indicators using data preprocessing, statistical analysis, and visualization techniques to better understand behavioral and demographic patterns within the dataset.

The analysis includes:

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Statistical analysis
* Data visualization
* Insight generation

## Features

* Mental health trend analysis
* Exploratory data analysis
* Data preprocessing and cleaning
* Statistical visualization
* Insight extraction from healthcare-related data

## Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook

## Objectives

* Explore mental health-related patterns in the dataset
* Apply exploratory data analysis techniques
* Generate visual insights from healthcare data
* Support data-driven analysis through statistical methods

## Project Workflow

Dataset → Data Cleaning → Exploratory Analysis → Visualization → Insights

## Results

The project provides analytical and visual insights that help identify patterns and relationships within mental health-related data.
